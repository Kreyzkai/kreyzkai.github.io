📚 How to Use Word Guesser Maker

🎮 Game Setup

  • Enter a Game Title for your word game
  • Set Mistakes Allowed (1-10) - how many wrong guesses before game over

❓ Adding Words

  • Word: The word students must guess (supports spaces, punctuation, numbers)
  • English Clue: Hint in English
  • Japanese Clue: Hint in Japanese (optional)
  • Students can toggle between English/Japanese clues during play
  • Click ✖ to delete a word

🔗 Linking Games (Next Game Feature)

  • Check Link to another game after completion
  • Select the next game type from dropdown
  • Upload the .html file of the next game
  • Embedded directly - works offline!

📤 Upload & Download

  • 📤 Upload Game: Load a previously saved game to edit
  • 💾 Download: Generate the student game file
  • 👀 Preview Game: Test before downloading

👨‍🎓 Student Experience

  • Click letters on keyboard to guess
  • Toggle between English/Japanese clues with radio buttons
  • ❤️ Hearts show remaining lives
  • Green = correct letter, Red = wrong letter
  • 🔍 Zoom button adjusts text size
  • Navigate with ⬅ Back, 🔄 Retry, Next ➡

💡 Auto-Add Feature

Start typing in the last word row and a new empty row appears automatically!
