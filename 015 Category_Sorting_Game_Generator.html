How to Use Tool 015

Step 1 — Game Settings

Enter a title and choose how many categories (2–6).

Step 2 — Category Names

Give each category a clear name, e.g. "Mammals", "Reptiles", "Birds".

Step 3 — Add Items

  1. Click + Add Item to add a slot.
  2. Optionally upload an image by clicking the image box.
  3. Optionally type a label. At least one of image or label is required.
  4. Choose which category this item belongs to.
  5. Every category needs at least one item.

Step 4 — Next Game (optional)

Choose another game HTML file. Students can jump to it from the congrats screen.

Step 5 — Download

Click Download Student Game. The file works offline.

Load a Saved Game

Click Load Saved Game to restore settings from a previously downloaded file.

Student Gameplay

  • Desktop: drag cards into category boxes.
  • Touch: tap a card to select it (glows blue), then tap a category.
  • Wrong = red flash, snaps back. Correct = card stays, turns green.
  • Hover over an image card and click 🔍 to view it full-size.
  • 🔍 button in bottom-right corner cycles through 5 text sizes.
