About This Site

Teaching Tools — CMarasigan for Kasai-Minami High School

A private collection of browser-based tools for generating classroom games and worksheets. Everything runs entirely in your browser — no internet connection needed once the files are loaded, and no data is ever sent anywhere.

What each tool does
001Multiple-choice quiz generator — text only
002Two-option (A/B) quiz with optional image uploads
003Hangman-style word guessing game
004Sentence word-order jumbling game
005Fill-in-the-blank printable worksheet
006Multiple-choice quiz with image answer choices
007Enhanced jumbler — words, phrases, or full sentences
008Interactive fill-in-the-blank game with scoring
009Full Jeopardy game generator with animations
010Two-column translation worksheet (beta)
011Basic Jeopardy — simpler, faster to set up
012Custom card deck — printable or on-screen
013PPTX text extractor — animation order, fully offline
014Multi-set drag-and-match game with text/image support, review screen, and built-in A4 worksheet + RTF export
015Category sorting game — create categories and sortable entries
016Watching Worlds FITB — fill-in-the-blank generator specific to the Watching Worlds textbook
017Multiple-choice quiz builder with reusable answer choices across quizzes
018AI image generator powered by Flux — text-to-image (beta)
019Media format converter — converts common audio and video formats (beta)
020Text-to-speech with preloaded ElevenLabs voices (beta)
021Voice clone TTS — record your voice to create a clone via Fish Audio, then generate speech in it. Free tier (beta)
022Falling Words Generator — teacher enters JP/EN pairs, exports an arcade-style student vocabulary game
023OCR text extractor — pulls readable text from uploaded or pasted images, supports Japanese and English
How to use

Click any card to open that tool. Fill in your content, preview, and download the student file. Student files are standalone HTML — just open and play, no installation needed.
