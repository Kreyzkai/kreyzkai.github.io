📚 How to Use Multiple Choice Maker

🎮 Game Setup

  • Enter a Game Title (e.g., "Comet II Lesson 11")
  • This will appear on the student game intro screen

❓ Adding Questions

  • Each row represents one question with multiple choices
  • Question: Enter the question text
  • Hint: Optional clue to help students (they can toggle it on/off)
  • Correct Choice: The right answer (always goes in the first choice box)
  • Choice 2-4: Additional answer options (wrong answers)
  • Image: Optional visual aid for the question
  • Click ✖ Delete to remove a question

✨ Text Formatting

  • **Bold text** → Bold text
  • *Italic text* → Italic text
  • __Underlined text__ → Underlined text
  • Press Enter in questions or hints to create line breaks
  • Use formatting to emphasize key words or create multi-part questions!

🎯 How Choices Work

  • The first choice is always the correct answer
  • Choices 2-4 are distractors (incorrect options)
  • You can have 2-4 total choices per question
  • The student game will automatically shuffle the choices randomly
  • Students won't know which position is correct!

🖼️ Adding Images

  • Click the file input in the rightmost column
  • Select an image from your computer
  • A small preview will appear below the file input
  • Images are embedded in the downloaded game file

📤 Upload & Download

  • 📤 Upload Game: Load a previously downloaded game HTML file to edit it
  • 💾 Generate Student Game: Download your quiz as an HTML file
  • Downloaded games preserve all questions, hints, choices, and images
  • You can upload and edit games as many times as needed

👨‍🎓 Student Experience

  • Students see one question at a time with shuffled answer choices
  • They can click "Show Hint" if you provided one
  • Green checkmark = correct answer
  • Red X = incorrect, and the correct answer is highlighted in green
  • Use ⬅ Back and Next ➡ buttons to navigate between questions
  • 🔍 Zoom button adjusts text size for better readability

💡 Auto-Add Feature

When you start typing in the last question row, a new empty row automatically appears below it. This makes it quick to add multiple questions without clicking "add" buttons!
