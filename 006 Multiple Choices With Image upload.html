📚 How to Use Image Multiple Choice Maker

🎮 Game Setup

  • Enter a Game Title / Topic for your quiz
  • This appears at the top of the student game

❓ Adding Questions

  • Question: Enter the main question text
  • Clue: Optional hint (students click the question to reveal it)
  • Each question has 4 choice slots with optional images and text
  • Tick the radio button next to the correct answer
  • Click ✖ to delete a question

🖼️ Adding Images to Choices

  • Click "Upload" under any choice to add an image
  • Images can be standalone or combined with text labels
  • Preview appears immediately after upload
  • Perfect for visual identification quizzes (animals, flags, objects, etc.)

✨ Text Formatting

  • **Bold text** → Bold text
  • *Italic text* → Italic text
  • __Underlined text__ → Underlined text
  • Press Enter in questions/clues for line breaks

🔗 Linking Games (Next Game Feature)

  • Check Link to another game after completion
  • Select the next game type from dropdown
  • Upload the .html file of the next game
  • Embedded directly - works offline!

🎨 Choice Colors

  • Choice 1 = Blue
  • Choice 2 = Red
  • Choice 3 = Green
  • Choice 4 = Yellow

📤 Upload & Download

  • 📤 Upload Game: Load a previously saved game to edit
  • 💾 Save & Download: Generate the student game file
  • 👀 Preview Game: Test before downloading

👨‍🎓 Student Experience

  • Click the question text to toggle between question and clue
  • Colored buttons with images and/or text for each choice
  • Correct answer expands to fill the screen
  • Wrong answers fade but can retry others
  • 🔍 Zoom button adjusts text size
  • Navigate with ⬅ Back, 🔄 Retry, Next ➡

💡 Auto-Add Feature

Start typing in the last question row and a new empty row appears automatically!
