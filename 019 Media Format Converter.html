📖 How to Use

Step-by-step

  • Drop one or more files onto the zone, or click to browse
  • Each file appears in the queue — change its output format if needed
  • For audio files, set the global quality (128 / 192 / 320 kbps)
  • Click Convert All and wait — images are instant, audio is seconds, video is slower
  • Click Download ZIP to get all converted files in one archive

Supported Formats

  • Audio: M4A, MP3, WAV, OGG, FLAC, AAC, OPUS, WMA → MP3 / WAV / OGG / FLAC / AAC / OPUS
  • Video: MP4, MOV, AVI, MKV, WebM, WMV → MP4 / WebM / AVI / MKV, or extract audio as MP3
  • Image: PNG, JPG, WEBP, BMP, GIF → JPG / PNG / WEBP (instant, no engine needed)

Notes

The FFmpeg engine loads once on first audio/video conversion. Video re-encoding is slow in the browser; use "MP3 (audio only)" for quick audio extraction from video.
