📖 How to Use

Steps

  • Drop one or more files — or click to browse. Multiple files OK.
  • Each file shows in the queue with its output format. Change it if needed.
  • Set audio quality (128 / 192 / 320 kbps) in the bar above.
  • Click Convert All. Images convert instantly; audio takes a few seconds; video is slowest.
  • Click Download ZIP when done.

Audio (no engine needed ✅)

  • Input: M4A, MP3, WAV, OGG, FLAC, and most audio formats
  • Output: MP3 (via JS encoder), WAV (pure JS)

Image (no engine needed ✅)

  • PNG / JPG / WEBP / BMP / GIF → JPG / PNG / WEBP

Video (loads FFmpeg ⚠️)

  • MP4, MOV, AVI, MKV, WebM → MP4 / WebM / AVI / MKV
  • Or extract audio track as MP3 (faster than full re-encode)
  • Requires a one-time ~25 MB download. Slow in browser — short clips only.
