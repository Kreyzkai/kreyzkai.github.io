📚 How to Use Fill-in-the-Blank Maker

🎮 Game Setup

  • Enter a Game Title for your activity
  • Choose Display Mode:
    • 📄 Full Paragraph - Show all text at once
    • 📝 Sentence-by-Sentence - Progress through one sentence at a time
  • Set Difficulty:
    • Easy - Word bank with all correct answers
    • Medium - Show first letter hints
    • Hard - Type answers (no word bank)
  • Set Distractors (0-10) - wrong answer choices to make it harder

📝 Adding Content

  • Paste or type your text in the text area
  • Click Next: Select Words & Phrases
  • Click individual words to make them blanks
  • Click and drag to select multi-word phrases as blanks
  • Click Clear All to start over

👁️ Preview & Download

  • Preview Game - Test the game before downloading
  • 📥 Download for Students - Save the student game file

📤 Load Existing Game

  • Upload a previously downloaded game to edit it
  • All settings, text, and blank selections are restored
  • Make changes and download again!

🔗 Linking Games (Next Game Feature)

  • Check Link to another game after completion
  • Select the next game type from dropdown
  • Upload the .html file of the next game
  • Embedded directly - works offline!

👨‍🎓 Student Experience

  • Students see blanks where you selected words
  • In Easy mode: click words from word bank to fill blanks
  • In Medium mode: type answers with first letter hints
  • In Hard mode: type complete answers
  • Instant feedback - green = correct, shake = try again

💡 Pro Tips

  • Select key vocabulary words for focused practice
  • Use multi-word phrases for idiomatic expressions
  • Sentence mode is great for longer texts
  • Add distractors for vocabulary discrimination practice
