📖 How to Use

🆕 Building a Game

  • Fill in the Game Title and Subject.
  • Click ＋ Add Set for each round of the game.
  • Each set has an Instruction students see before playing, plus Left and Right column labels.
  • Add at least 3 entries per set (English + Japanese required; images optional).
  • Click ⬇ Download Student Game to save.

📂 Loading a Previous Game

  • Upload any previously downloaded Match Game .html to restore all sets.

🔗 Linking a Next Game

  • Upload another game's .html file. A Next Game → button appears on the final end screen — fully offline.

🎮 How Students Play

  • Choose English or Japanese labels, then click Start.
  • Match word cards to image/text cards. Wrong picks flash red.
  • After completing a set, click Review Answers to see correct pairs.
  • Then click Next Set → to continue.
  • Use the 🔍 zoom button to adjust text size.
