📚 How to Use Choices Game Maker

🎮 Game Setup

  • Customize the Choice Labels (True/False, Yes/No, A/B/C/D, etc.)
  • You can use 2, 3, or 4 choices - leave optional labels blank if not needed
  • Labels appear on all questions throughout the game

❓ Adding Questions

  • Question/Statement: Enter the question or statement text
  • Correct Answer Dropdown: Select which choice label is correct
  • Image: Optional visual for the question
  • Click ✖ Delete to remove a question

✨ Text Formatting

  • **Bold text** → Bold text
  • *Italic text* → Italic text
  • __Underlined text__ → Underlined text
  • Press Enter in questions to create line breaks

🔗 Linking Games (Next Game Feature)

  • Check Link to another game after completion to chain games
  • Select the Next Game Type from the dropdown
  • Upload the .html file of the next game
  • The next game is embedded directly - works offline!

🎨 M&M Choice Colors

  • Choice 1 = Blue M&M
  • Choice 2 = Red M&M
  • Choice 3 = Green M&M
  • Choice 4 = Yellow M&M

📤 Upload & Download

  • 📤 Upload Game: Load a previously downloaded game to edit it
  • 💾 Download Student Game: Save your game as an HTML file
  • 👀 Preview Game: Test the game before downloading

👨‍🎓 Student Experience

  • Students see colored choice buttons (like M&Ms!) for each question
  • Correct answer shows green checkmark ✅
  • Wrong answers grey out but can retry other options
  • Navigate with ⬅ Back, 🔄 Retry, Next ➡ buttons
  • 🔍 Zoom button adjusts text size (bottom bar stays fixed!)

💡 Auto-Add Feature

When you start typing in the last question row, a new empty row automatically appears!
